# Taberna do Mafra

A **Taberna do Mafra** é uma plataforma de e-commerce voltada ao universo de RPGs de Mesa. O objetivo é oferecer uma experiência imersiva, simulando a atmosfera de uma taberna clássica por meio de uma loja moderna.

O site atende dois públicos principais:
* **Fantasia Medieval:** Jogadores de D&D, Tormenta20 e Pathfinder.
* **Horror Contemporâneo:** Jogadores de Ordem Paranormal, Vampiro: A Máscara e Call of Cthulhu.

## Identidade Visual
A interface foi pensada para ser acessível e temática, utilizando a seguinte paleta de cores:
* **Fundo (#FAFAF5):** Tom off-white simulando papel/pergaminho para não cansar a vista.
* **Estrutura (#2C3E50):** Azul Navy para rodapés e cabeçalhos.
* **Ação (#CC5500):** Laranja para botões de compra e links importantes.

## Estrutura do Site
* **index.html:** Tela inicial com banner e vitrines de produtos (Promoções e Mais Vendidos).
* **catalogo.html:** Lista completa com todos os produtos disponíveis.
* **detalhes.html:** Página de visualização de produto.
* **sobre.html:** Página institucional com a missão e história da loja.
* **atendimento.html:** Formulário de contato e canais de suporte.

## Tecnologias Utilizadas
* **HTML5 Semântico**
* **CSS3**
* **Google Fonts:** Fontes *Merriweather* (títulos) e *Open Sans* (corpo do texto).

## Como Rodar
1.  Baixe a pasta do projeto.
2.  Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox).
